# Eventos con NestJS

Los eventos en NestJS siven para desacoplar responsabilidades.

## Ejecutar del proyecto

1. Clonar el repositorio
2. Instalar el proyecto

```
yarn install
```

3. Ejecutar el produccion

```
yarn build
```
